# Обрезка ссылок с помощью вк
Данный скрипт представляет собой консольное приложение на Python, которое сокращает ссылки с использованием API VK. Утилита позволяет пользователю вводить URL-адрес, получая ее сокращенный вариант, если же ссылка уже скоращена - получить статистику по количеству кликов по ним.

# Как установить
Операционная система: любая, где доступен Python 3 (Windows, macOS, Linux). Используем 'pip' для установки зависимостей (requests, python-dotenv):
```
pip install - r requirements.txt
```
Рекомендуется использовать virtualenv/env для изоляции проекта.
Создадим файл .env и добавим токен ВКонтакте.

# Примеры запуска скриптов
Сокращение ссылки выглядит следующим образом:
```
python main.py https://dvmn.org/modules/web-api/lesson/migration-from-website/
Сокращенная ссылка: https://vk.cc/cEAhrC
```
Получение статистики по кликам:
```
python main.py https://vk.cc/cEAhrC
Количество кликов по ссылке: 4
```

# Переменные окружения
Программа использует нестандартные переменные окружения для настройки. Эти переменные не могут быть автоматически обнаружены и должны быть указаны в файле `.env` перед запуском программы.

**VK_IMPLICIT_FLOW_TOKEN** - токен доступа к API ВКонтакте, необходимый для выполнения запросов к API для сокращения ссылок и получения статистики по кликам. Форматом является строка (например, `1234567890abcdef1234567890abcdef`). Без этого токена функции, связанные с API запросами, не будут работать, и программа завершит выполнение с ошибкой.
